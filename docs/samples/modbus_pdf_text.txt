1.1.1 Introduction to Modbus TCP Connector
Welcome to the Modbus TCP Connector documentation for Industrial Edge.
The Modbus TCP Connector connects various devices via the Modbus TCP protocol to the
Industrial Edge system.
Connectors collect data from PLCs and devices and make it available on the Industrial Edge
system. The Modbus TCP Connector accesses data via the Modbus TCP protocol.
This connector is part of the Connectivity Suite (CS) and supports standardized conﬁguration
and data transfer.
The Common Conﬁgurator handles the connector conﬁguration. It is available free of charge in
the Industrial Edge marketplace.
Industrial Edge App developers and users can integrate the data into their own applications and
services. The Industrial Edge Connectors provide a uniﬁed data interface that reduces
development eﬀort and forms the basis for modularized software solutions.
The Modbus TCP Connector supports reading and writing values for devices via the Modbus TCP
protocol. The following devices are supported:

Manual, 10/2025, A5E52905450-AD 153.1 Overview of Modbus TCP Connector
Modbus TCP Connector is an app that runs on IE Devices. Install Modbus TCP Connector on IE
Devices to connect to your Modbus TCP PLCs.
Modbus TCP Connector transfers tag values from Modbus TCP PLCs to the Databus or Industrial
Information Hub (IIH). This enables you to analyze and visualize Modbus TCP tag values in apps
such as Performance Insight.
Modbus TCP Connector supports any Modbus TCP PLC that uses the standard protocol.
As a Connectivity Suite (CS) Connector, Modbus TCP Connector also makes tag values available
directly in the IIH without using the Databus. For more information, see IIH documentation.

A data source is a device that provides data. Use the Common Conﬁgurator to conﬁgure Modbus
TCP Connector to consume data from devices on your IE Device.
6.1.1.1 Accessing the Modbus TCP Connector page
Follow these steps to access the Modbus TCP Connector conﬁguration page:
1. Launch your IE Device and navigate to "Apps > Common Configurator".
2. Click "Get Data > Connector Configuration".
The "Get Data" page opens and displays all available connectors.
